# Anforderungen – Gamification Rollenspiel

## Muss-Anforderungen

### Spielmechanik
- [M1] Das Spiel besitzt eine steuerbare Spielfigur.
- [M2] Die Spielfigur kann zwischen mehreren Orten (z. B. Räume/Höhlen) navigieren.
- [M3] Orte repräsentieren Themenbereiche der IHK-Abschlussprüfung.

### Punktesystem
- [M4] Für jeden Themenbereich wird ein Punktestand geführt.
- [M5] Das Spiel kann nur beendet werden, wenn in jedem Themenbereich eine Mindestpunktzahl erreicht wurde.

### Belohnungen
- [M6] Es existieren Belohnungen (z. B. Bronze, Silber, Gold) abhängig von der Güte der Aufgabenbewältigung.

### Persistenz
- [M7] Spielstände können gespeichert und geladen werden.

### Fragenkatalog
- [M8] Mindestens 40 automatisch auswertbare Fragen zu einem ausgelosten Themenbereich sind vorhanden.
- [M9] Unterstützte Fragetypen: Multiple Choice, Wahr/Falsch, Lückentext.
